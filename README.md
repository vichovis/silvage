# Silvage

> Discover new purposes for old hardware.

Silvage is an open-source platform for discovering, documenting and sharing ways to repurpose existing hardware.

Instead of focusing only on specifications, benchmarks or compatibility, Silvage connects hardware with real-world projects built by the community.

Whether it's an old desktop, a laptop rescued from storage or an enterprise server, the goal is to answer a simple question:

> **What can this hardware become?**

## Goals

Silvage aims to:

- Document hardware and real-world configurations.
- Help users find practical uses for existing hardware.
- Build an open, community-maintained knowledge base.
- Extend the useful life of computers and reduce electronic waste.

## Roadmap

### Phase 1

- Hardware database
- Hardware pages
- Project pages
- Search
- Basic user accounts

### Phase 2

- Community submissions
- Hardware compatibility reports
- Operating system support
- Performance reports
- Upgrade recommendations

### Phase 3

- Hardware detection CLI
- Public API
- Collections
- Project galleries
- Advanced search and filtering

## Tech Stack

The initial implementation is planned with:

- SvelteKit
- FastAPI
- PostgreSQL (Supabase)
- Docker

This stack may evolve as the project grows.
